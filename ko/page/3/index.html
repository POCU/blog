    안녕하세요, 수강생 여러분! 2021년 8월 13일부터 깃헙(GitHub)에서 Git 작업을 인증할 때 더 이상 계정 비밀번호를 이용할 수 없습니다. 그 대신 토큰 기반 인증을 사용해야 한다고 하는군요. 그래서 이번 포스팅에서는 토큰 기반 인증 중 하나인 **개인용 접근 토큰(PAT, Personal Access Token)**을 사용하는 방법을 설명드리겠습니다. 이 포스트에서는 Windows 기반의 PC에서 Chrome 브라우저를 사용할 때를 기준으로 설명합니다.
