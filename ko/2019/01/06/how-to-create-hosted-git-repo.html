가이드: 웹 기반 비공개 Git 가입하고 저장소 만들기

문서봇 2019-01-06
  • 첫주
  • GIT
  • 가이드

이 가이드에서는 웹 기반의 비공개 Git 저장소를 만드는 방법을 설명합니다.

POCU 아카데미에서 지원하는 Git 저장소 서비스는 아래와 같습니다.

  • 깃허브(GitHub): 무료
  • 깃랩(GitLab): 무료

깃허브

첫 번째로 깃허브에서 비공개 Git 저장소를 만드는 방법을 안내하겠습니다.

깃허브 계정 만들기

깃허브에서 Git 저장소를 만드려면 회원 가입을 해야합니다.

  1. 깃허브 홈페이지에 들어가서 우측 상단의 Sign Up 버튼을 누릅니다.
  2. Username, Email address, Password를 입력하고, 퍼즐까지 풀고 Create an account를 누릅니다.
  3. 비공개 저장소를 사용하기 위해 Choose your personal plan에서 Free(무료)를 선택하고 Submit을 누릅니다.

비공개 저장소 만들기

회원 가입을 하셨다면, 이제 저장소를 만들 차례입니다.

  1. 깃허브 홈페이지에 로그인합니다.
  2. New 혹은 Create a new repository 버튼을 누릅니다.
  3. Repository name을 자유로이 입력합니다.
  4. Public과 Private 중 꼭 Private을 선택해야 됩니다.

깃랩

다음은 깃앱에서 비공개 Git 저장소를 만드는 방법입니다.

깃랩 계정 만들기

깃랩 역시 Git 저장소를 만들기 위해서는 회원 가입이 필요합니다.

  1. 깃랩 홈페이지에 갑니다.
  2. 오른쪽 상단 구석의 Sign in를 누릅니다.
  3. Registser now 링크를 누릅니다.
  4. 가입하는 방법은 두 가지가 있는데 편하신 방법을 따라 해 주세요.
    • 소셜 로그인을 통해 가입하기
      1. 아래로 스크롤합니다.
      2. 원하는 소셜 로그인을 선택해서 진행합니다.
      • 이 경우 로그인 후에 Settings > Account에서 Username을 확인하고, Settings > Password에서 Password를 만들어야 합니다.
    • 직접 가입하기
      1. 프라이버시 보호 차원에서 Full name은 아무 이름이나 입력하셔도 됩니다.
      2. Username, Email, Password는 제대로 입력합니다.
      3. 아래 Register 버튼을 누릅니다.
  5. 화면에 나오는 설명에 따라 가입을 완료하세요.

비공개 저장소 만들기

회원 가입을 하셨다면, 이제 저장소를 만들 차례입니다.

  1. 깃랩 홈페이지에 로그인합니다.

  2. 우측 상단에 Create New > New project/repository를 누릅니다.

  3. Create Blank project/repository 탭을 선택합니다.

  4. Project name을 입력합니다. 마음에 드는 이름을 넣어 주세요.

  5. Visibility Level은 꼭 Private을 선택해야 됩니다.

  6. Create project를 누릅니다.
